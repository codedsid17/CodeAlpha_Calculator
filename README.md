# 🐾 Cat Calculator

A cute pixel-art themed calculator built using HTML, CSS, and JavaScript. Inspired by cozy retro aesthetics and adorable cat companions, this calculator transforms everyday calculations into a fun and charming experience.

## ✨ Features

- Basic arithmetic operations
- Responsive design
- Pixel-art inspired interface
- Cute cat-themed UI
- Interactive button effects
- Clean and user-friendly layout

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
